# Cookbook App

A recipe sharing social media app built with Ruby on Rails. Users can post recipes, browse others' cookbooks, and save their favorites.

## Tech Stack

- **Ruby on Rails**
- **Devise** — authentication
- **CarrierWave + MiniMagick** — image uploads
- **Pagy** — pagination
- **Friendly ID** — clean URLs


Visit `http://localhost:3000`
